# Cookie Clicker for Casio Graph Math+ Calculator
A simple and fun Cookie Clicker game designed to run on a Casio Graph Math+ calculator.

## Description
This project is a simplified version of the famous Cookie Clicker game, adapted to be played on a Casio Graph Math+ calculator. The goal is to collect as many cookies as possible by clicking and purchasing upgrades.

## Features
- **Cookie Collection:** Press the OK key to earn cookies.
- **Upgrades:** Purchase upgrades to increase your cookie production.
- **Achievements:** Unlock achievements by reaching certain milestones.
- **User Interface:** Display statistics and available options directly on the calculator screen.

## Requirements
A **Casio Graph Math+ calculator.**

Required libraries to run Python scripts on the calculator: **casioplot** and **turtle**.

## Installation
1. Download the cookie.py and donnees.py files from this repository.
2. Transfer them to your calculator.
3. Run the cookie.py file to start the game.

## How to Play
Start the Game: Press any key to begin.
Collect Cookies: Press `OK` to earn cookies.
Buy Upgrades: Use keys `1`, `2`, and `3` to purchase upgrades.
View Achievements: Press `CATALOG` to see your achievements.
Access Settings: Press `TOOLS` to access settings.
Quit the Game: Press `RETURN` to exit.
